Find the hidden message

Let's find the very first block in the bitcoin blockchain. Click the button below to open a block explorer exactly at block 0, which is called the genesis block.

Scroll down and expand the details on the one transaction that is stored in this block. Find the input called “Coinbase”. Now look for the label “SCRIPTSIG ( HEX )”. The value next to it is an encoded message.

Copy that value and paste it in the code block.
